# 🏠 城市房产价格趋势透视看板 (Real Estate Data Dashboard)

本应用是一个基于 **Streamlit** 构建的**城市房产价格数据可视化看板**，专注于展示中国核心城市在过去多年间的**房价和房租趋势**。通过交互式筛选和 Altair 图表，用户可以轻松对比不同区域的价格涨幅和走势。

-----

## ✨ 核心功能与亮点

  * **📈 房价/房租切换：** 用户可以一键切换数据视角，查看特定城市区域的**平均二手房挂牌价格**（房价）或**平均租赁价格**（房租）的长期走势。
  * **📍 区域精细对比：** 支持在选定的城市内，选择多个**城区/区域**进行叠加对比，清晰洞察不同区域的相对表现。
  * **📅 灵活时间筛选：** 通过时间滑块自由选择分析时间段，聚焦短期波动或长期增长趋势。
  * **📊 涨幅指标榜：** 自动计算所选时间区间内的**百分比涨幅**，并以卡片形式展示区域间的增长率排名。
  * **技术栈优化：** 采用 **Pandas** 进行高效数据处理，**Altair** (基于 Vega-Lite) 生成美观且完全交互式的响应式图表。

## 🛠️ 技术细节

### 数据处理 (`get_gdp_data` 函数)

应用使用 `pd.melt` 将宽格式（每年一列）的数据转换为长格式（价格和时间作为列），这极大地简化了 Altair 的数据绑定和时间序列分析。

### 图表渲染 (Altair)

图表使用了 **Altair** 库，该库专为声明式数据可视化设计。关键代码实现了：

1.  `alt.Chart`: 定义基础数据源和 X/Y/Color 编码。
2.  `lines` + `points`: 将折线图和交互式圆点（Tooltip）叠加，提供清晰的视觉效果和数据悬停详情。
3.  `.interactive()`: 启用图表的缩放和平移功能。
